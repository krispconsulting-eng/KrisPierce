# Social template kit

`MasterClass_Executive_Kit.dc.html` is a design-doc canvas of post and reel
templates for announcing Kris's acceptance into the MasterClass Executive
founding cohort (square 1:1 and portrait 4:5 layouts). `support.js` and
`imageslot.js` are the runtime the canvas needs to render outside its
original editor — open the `.dc.html` file in a browser to view it.
