## Rare Diseases NSW Co-Design Workshop

A facilitator deck for running a co-design workshop with the rare disease
community in NSW. It walks participants through an established systemic
co-design framework, tests it against the reality of rare disease, and agrees
how the group will work together.

- **Deck:** [`Rare-Diseases-NSW-Co-Design-Workshop.pptx`](Rare-Diseases-NSW-Co-Design-Workshop.pptx) — 21 slides, 16:9.
- **Generator:** [`build_deck.py`](build_deck.py) — regenerate or edit content, then
  `pip install python-pptx && python3 build_deck.py`.

**Flow:** aim of the day → suggested approach → the current framework
(principles & values) → *discussion: do these make sense? what's missing for
rare disease?* → our Code of Care → the four domains (People · Process ·
Practice · Place) including key teams, roles and Shades of Co-Design → the
co-design maturity journey → first steps.

**Framework credit:** adapted from Emma Blomkamp's *Systemic Design Practice*
and New Know How (Shades of Co-Design, Co-Design Teams, Co-Design Maturity
Model). CC BY-NC.
