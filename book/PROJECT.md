## Chapter production pipeline

Every chapter moves through, in order:

**Idea → Outline → Scene list → First draft → Reflection → Revision → Review → Approved**

Track each chapter's stage in `08-chapters/00-architecture.md`.

## The Story Engine

Every chapter answers four questions:

1. **What happened?** — the event.
2. **What changed?** — the transformation.
3. **Why does it matter?** — the universal lesson.
4. **Why now?** — why this belongs here, at this point in the book.

A chapter that can't answer all four isn't ready to draft.
