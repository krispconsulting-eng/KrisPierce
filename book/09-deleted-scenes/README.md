# Deleted Scenes

**Never delete. Move here.**

threads — lands in this folder with a one-line note on why it was cut and
where it might live next.

Many deleted scenes become:
- Keynote and conference talk material
- Website extended stories
- Podcast episode seeds
- Book-club workbook prompts
